[[2. Тип данных date]]
```python
from datetime import date
my_date = date(1992, 10, 6) # тип date: год + месяц + день 
```

Используется функция `date()`

**Замечание:** Конструктор типа `date` сначала принимает год, затем месяц, а уже потом день. Мы также можем использовать именованные аргументы, нарушая указанный порядок 

```python
date(day=6, month=10, year=1992)
```
