Для этого используется **интерактивный режим добавления изменений** в функции [[git add]] (`git add -p`). Вот пошаговая инструкция:

	git add -p <имя_файла>  # Для конкретного файла
или можно:
	`git add -p   # Для всех изменённых файлов`

Git будет показывать вам каждый блок изменений (hunk) и спрашивать:

	Stage this hunk [y,n,q,a,d,s,e,?]?
